Artwork/Project Description A 3D game, made using Unity Engine to explore Computer Engineering Department of Sepuluh Nopember Institute of Technology in Virtual Reality. This is a group project for Telematics Project course. The project is showcasing various facilities and laboratories. The game allows users to explore the campus in a 3D environment, providing an immersive experience for prospective students and visitors. I was responsible for developing the logic and mechanics of the game, including the interaction system and user interface.
